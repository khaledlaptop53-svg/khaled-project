# University Registration System 🎓

This project is an interactive university registration system built with Python, applying advanced OOP concepts like Abstraction, Inheritance, and Encapsulation.

## ✨ Key Features
- *Course Management:* Handles both Lectures and Labs.
- *Tuition Calculation:* Automated billing with tax and equipment fees.
- *Student Schedule:* Add/Remove courses with validation to prevent duplicates.
- *Input Validation:* Ensures data integrity (e.g., credit hours and costs).
